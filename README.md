# PromptRadar Mini

PromptRadar Mini 是一个轻量网页 APP，用于管理个人 AI 提示词，并记录每日发现的提示词线索。第一阶段不接入后端、数据库、登录系统或外部平台 API，所有数据都保存在浏览器本地。

## 项目用途

- 保存和管理常用 AI 提示词。
- 按场景、标签、收藏状态快速筛选提示词。
- 记录从文章、社区、搜索关键词或人工观察中发现的提示词线索。
- 将值得收藏的雷达内容转入本地提示词库。
- 将个人提示词库导出为 Markdown 或 CSV，便于备份和沉淀到知识库。

## 当前功能

- 我的提示词库：新增、编辑、删除、搜索、复制提示词。
- 提示词字段：标题、场景、正文、标签、备注、收藏状态、创建时间、更新时间。
- 场景筛选：写作、翻译、编程、图片生成、视频生成、办公、学习、资料查询、其他。
- 收藏筛选：支持只看收藏。
- 排序规则：提示词列表默认按更新时间倒序排列。
- 复制功能：支持复制提示词正文和复制完整卡片。
- 导出功能：支持 Markdown 和 CSV，CSV 带 UTF-8 BOM，适配 Excel/WPS 中文显示。
- 每日提示词雷达：支持状态管理、状态筛选、手动新增来源内容、转入提示词库。
- 雷达状态：未处理、值得收藏、已忽略、已收藏。
- 本地持久化：提示词库和雷达内容都保存到 localStorage。
- 响应式布局：适配电脑和手机。

## 本地运行方法

```powershell
npm.cmd install
npm.cmd run dev -- --host 127.0.0.1
```

启动后访问：

```text
http://127.0.0.1:5173/
```

如果你的系统允许直接运行 npm，也可以使用：

```powershell
npm install
npm run dev
```

## 构建方法

```powershell
npm.cmd run build
```

构建产物会生成在 `dist/` 目录。

## 数据保存说明

当前版本没有后端和数据库，数据保存在浏览器 localStorage 中：

- 提示词库：`promptradar-mini-prompts`
- 雷达内容：`promptradar-mini-radar-items`

清理浏览器站点数据、更换浏览器或更换域名/端口可能导致本地数据不可见。建议定期使用 Markdown 或 CSV 导出功能备份。

## 当前限制

- 不支持账号登录和多设备同步。
- 不支持云端备份和数据库检索。
- 不接入 YouTube、抖音、B站等外部平台 API。
- 雷达内容由内置示例和用户手动添加组成，不会自动抓取外部内容。
- localStorage 容量有限，不适合保存超大规模提示词库。

## 后续扩展方向

- 增加导入 Markdown/CSV 功能。
- 增加提示词模板变量和一键填充。
- 增加更细的标签管理和批量操作。
- 增加本地备份文件恢复。
- 增加可选的后端同步和多设备访问。
- 在用户明确授权后，再考虑接入外部平台或搜索 API。
