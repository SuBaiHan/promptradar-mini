# PromptRadar Mini

PromptRadar Mini 是一个轻量网页 APP，用于管理个人 AI 提示词，并记录每日发现的提示词线索。当前版本继续保持纯前端实现，不接入后端、数据库或登录系统，所有数据都保存在浏览器本地。

## 项目用途

- 保存和管理常用 AI 提示词。
- 按场景、标签、收藏状态快速筛选提示词。
- 手动收集来自 YouTube、B站、抖音、小红书、知乎、网页或其他来源的提示词线索。
- 手动搜索 YouTube 视频结果，并将有价值的内容保存到每日雷达。
- 对雷达内容进行价值评分、难度标注、状态管理和每日摘要整理。
- 将值得收藏的雷达内容转入本地提示词库。
- 将个人提示词库或今日雷达摘要导出为 Markdown/CSV，便于备份和沉淀到知识库。

## 当前功能

- 我的提示词库：新增、编辑、删除、搜索、复制提示词。
- 提示词字段：标题、场景、正文、标签、备注、收藏状态、创建时间、更新时间。
- 场景筛选：写作、翻译、编程、图片生成、视频生成、办公、学习、资料查询、其他。
- 收藏筛选：支持只看收藏。
- 排序规则：提示词列表默认按更新时间倒序排列。
- 复制功能：支持复制提示词正文和复制完整卡片。
- 导出功能：支持 Markdown 和 CSV，CSV 带 UTF-8 BOM，适配 Excel/WPS 中文显示。
- 每日提示词雷达：支持手动新增、批量导入、重复提示、状态筛选、平台筛选、日期筛选、关键词搜索和评分排序。
- 雷达字段：标题、来源平台、来源链接、搜索关键词、内容摘要、推荐价值评分、使用难度、适用场景、发现日期、状态。
- 雷达状态：未处理、值得收藏、已忽略、已收藏。
- 每日摘要：按当天发现日期统计，按平台分组，支持复制和导出 Markdown。
- YouTube 手动搜索：支持配置 API Key、关键词、结果数量和排序方式，点击按钮后调用 YouTube Data API v3 `search.list`。
- YouTube 搜索结果：展示视频标题、频道、发布时间、视频链接、搜索关键词和简介摘要，并可保存到每日雷达。
- 转入提示词库：自动带入标题、摘要、场景映射、平台和关键词标签，并保留原雷达记录。
- 本地持久化：提示词库和雷达内容都保存到 localStorage。
- 响应式布局：适配电脑和手机。

## 本地运行方法

```powershell
npm.cmd install
npm.cmd run dev -- --host 127.0.0.1
```

启动后访问：

```text
http://127.0.0.1:5173/
```

如果你的系统允许直接运行 npm，也可以使用：

```powershell
npm install
npm run dev
```

## 构建方法

```powershell
npm.cmd run build
```

构建产物会生成在 `dist/` 目录。

## 数据保存说明

当前版本没有后端和数据库，数据保存在浏览器 localStorage 中：

- 提示词库：`promptradar-mini-prompts`
- 雷达内容：`promptradar-mini-radar-items`
- YouTube 搜索设置：`promptradar-mini-youtube-settings`

YouTube API Key 只保存在当前浏览器 localStorage 中，不会写死在代码里。清理浏览器站点数据、更换浏览器或更换域名/端口可能导致本地数据不可见。建议定期使用 Markdown 或 CSV 导出功能备份。

## YouTube API Key 配置

1. 在 Google Cloud 中启用 YouTube Data API v3，并创建 API Key。
2. 打开“每日提示词雷达”页面，在“YouTube 手动搜索”区域填入 API Key。
3. API Key 输入框默认隐藏内容，可手动切换显示或隐藏。
4. 填写搜索关键词、结果数量和排序方式后，点击“搜索 YouTube”才会发起请求。
5. YouTube `search.list` 每次请求会消耗 API 配额，请避免无意义的重复搜索。

## 当前限制

- 不支持账号登录和多设备同步。
- 不支持云端备份和数据库检索。
- 仅支持 YouTube 手动搜索，不支持自动推送、定时搜索或后台任务。
- 不支持抖音/B站/小红书等平台自动搜索。
- 不支持云同步。
- 雷达内容由内置示例、批量粘贴、用户手动添加和用户手动触发的 YouTube 搜索组成，不会自动抓取外部内容。
- localStorage 容量有限，不适合保存超大规模提示词库。
